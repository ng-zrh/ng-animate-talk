Animation

Animation is the process of creating motion and shape change illusion by means of the rapid display of a sequence of static images that minimally differ from each other. The illusion — as in motion pictures in general — is thought to rely on the phi phenomenon. Animators are artists who specialize in the creation of animation. Wikipedia

ngAnimate

bower install angular-animate --save
angular.module('myApp', ['ngAnimate'])

Automagically enables animation support for core directives like ngIf, ngShow, ngClass and ngRepeat.

Three ways to perform animations: CSS3 Transitions, CSS3 Animations and JavaScript.

ngIf

Triggers: enter and leave

.fade { opacity: 1; } .fade.ng-enter, .fade.ng-leave { transition:0.5s linear opacity; } /* starting animations for enter */ .fade.ng-enter { opacity: 0; } /* terminal animations for enter */ .fade.ng-enter.ng-enter-active { opacity: 1; } /* starting animations for leave */ .fade.ng-leave { opacity: 1; } /* terminal animations for leave */ .fade.ng-leave.ng-leave-active { opacity: 0; }

ngClass

Triggers: add and remove of $my-class

.#{$my-class} { transform: scale(1.2); } .#{$my-class}-add, .#{$my-class}-remove { transition:0.5s linear transform; } /* starting animations for add */ .#{$my-class}-add { } /* terminal animations for add */ .#{$my-class}-add.#{$my-class}-add-active { } /* starting animations for remove */ .#{$my-class}-remove { } /* terminal animations for remove */ .#{$my-class}-remove.#{$my-class}-remove-active { }
